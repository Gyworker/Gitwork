## 1. 概述

### 1.1 补充背景
本方案针对需求文档V1.8和方案设计V1.0中遗漏的功能进行补充设计，包括：
- Excel通讯录导入
- 图片OCR文字识别
- 通讯录管理增强
- 映射学习功能增强

### 1.2 设计目标
1. 实现Excel文件导入功能，支持.xls/.xlsx格式
2. 实现图片OCR文字识别，支持名片、图片内容提取
3. 增强通讯录管理，支持导入导出Excel格式
4. 增强映射学习，支持从Excel文件学习映射关系
5. 所有导入数据可在后台保存为Excel文件并支持编辑

## 2. 功能详细设计

### 2.1 Excel通讯录导入

#### 2.1.1 功能描述
支持从Excel文件(.xls/.xlsx)导入通讯录数据，兼容多种列格式。

#### 2.1.2 输入格式
支持以下列格式（智能识别表头）：
| 列名 | 别名 | 说明 |
|------|------|------|
| 姓名 | name, 名称 | 必填 |
| 电话 | phone, 电话, 手机, mobile | 选填 |
| 邮箱 | email, 邮件, e-mail | 选填 |
| 公司 | company, 公司, 企业 | 选填 |
| 部门 | department, 部门, dept | 选填 |
| 职位 | position, 职位, title, 职务 | 选填 |
| 备注 | remark, 备注, note, notes | 选填 |

#### 2.1.3 数据处理
1. 读取Excel文件所有工作表
2. 智能识别表头行（自动检测表头位置）
3. 列名标准化映射
4. 数据清洗和验证
5. 重复数据检测（按姓名+电话）
6. 导入预览和确认

#### 2.1.4 后台存储
- 导入数据自动保存到 `data/imports/contacts_YYYYMMDD_HHMMSS.xlsx`
- 支持导出为标准Excel格式
- 文件可被外部编辑器编辑
- 重新导入时可读取已编辑的文件

#### 2.1.5 界面交互

### 2.2 图片OCR文字识别

#### 2.2.1 功能描述
使用Tesseract OCR引擎从图片中提取文字，支持名片识别、内容导入。

#### 2.2.2 支持的图片格式
- JPG/JPEG
- PNG
- BMP
- TIFF

#### 2.2.3 OCR处理流程
```
图片输入 → 预处理 → OCR识别 → 智能解析 → 结构化数据
```

#### 2.2.4 预处理步骤
1. 灰度转换
2. 对比度增强
3. 去噪声
4. 二值化
5. 倾斜校正

#### 2.2.5 名片识别解析
智能从OCR文本中提取：
- 姓名（通常是大字体）
- 电话号码（正则匹配）
- 邮箱地址（正则匹配）
- 公司名称
- 部门/职位
- 地址信息

#### 2.2.6 嵌入打包方案
Tesseract将捆绑打包到最终可执行程序中：

| 组件 | 路径 | 说明 |
|------|------|------|
| tesseract.exe | ./tesseract/ | OCR引擎主程序 |
| eng.traineddata | ./tesseract/tessdata/ | 英文语言包 |
| chi_sim.traineddata | ./tesseract/tessdata/ | 简体中文语言包 |
| 配置 | ./tesseract/tessdata configs/ | 识别配置 |

#### 2.2.7 后台存储
- OCR识别的图片保存到 `data/imports/ocr_images/`
- 识别结果保存为 `data/imports/ocr_results_YYYYMMDD_HHMMSS.xlsx`
- 支持导出为Excel格式
- 文件可被外部编辑器编辑

### 2.3 通讯录管理增强

#### 2.3.1 功能描述
增强通讯录管理功能，支持Excel导入导出、数据编辑。

#### 2.3.2 导出功能
- 导出全部通讯录为Excel
- 按条件筛选导出
- 导出模板下载

#### 2.3.3 批量操作
- 批量删除
- 批量修改
- 批量导入

#### 2.3.4 数据存储
- SQLite数据库存储核心数据
- Excel文件用于导入导出和数据备份
- 支持数据同步

### 2.4 映射学习增强

#### 2.4.1 功能描述
增强映射学习功能，支持从Excel文件学习映射关系。

#### 2.4.2 Excel映射学习
支持从Excel文件学习映射关系，格式要求：

| 列名 | 说明 |
|------|------|
| 关键词 | 模块/任务的关键词 |
| 责任人 | 对应的负责人姓名 |

#### 2.4.3 学习流程
```
导入Excel → 分析关键词 → 学习映射关系 → 保存映射库
```

#### 2.4.4 映射匹配算法
1. 精确匹配
2. 模糊匹配（包含关系）
3. 分词匹配
4. 相似度计算

```sql

-- 映射学习表
CREATE TABLE mapping_rules (
    id INTEGER PRIMARY KEY,
    keyword VARCHAR(200) NOT NULL,
    responsible VARCHAR(100),

    created_at DATETIME,
    updated_at DATETIME
);
```
