## 2. M7.数据管理 - 自动备份功能

### 2.1 功能概述

#### 2.1.1 功能名称
**自动备份功能** (M7.数据管理)

#### 2.1.2 功能目标
为市场咨询任务跟踪工具提供自动化的数据备份能力，确保业务数据的安全性和可恢复性。

#### 2.1.3 功能特性
- 定时自动备份
- 可配置的备份策略
- 旧备份自动清理
- 启动/关闭时自动备份
- 备份状态可视化

---

### 2.2 功能需求

#### 2.2.1 核心功能

**M7.1 自动备份服务**
- 单例模式运行的后台服务
- 支持多种备份频率
- 后台定时执行，不影响主程序运行
- 自动判断是否需要执行备份

**M7.2 备份策略配置**
| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| enabled | bool | false | 是否启用自动备份 |
| frequency | enum | daily | 备份频率 |
| backup_time | string | "02:00" | 每日备份时间 |
| keep_count | int | 7 | 保留备份数量 |
| backup_on_startup | bool | true | 启动时备份 |
| backup_on_shutdown | bool | true | 关闭时备份 |
| notify_on_backup | bool | true | 备份完成通知 |

**M7.3 备份频率选项**
| 频率 | 枚举值 | 说明 |
|------|--------|------|
| 每小时 | hourly | 每隔1小时备份一次 |
| 每4小时 | 4hours | 每隔4小时备份一次 |
| 每8小时 | 8hours | 每隔8小时备份一次 |
| 每天 | daily | 每天指定时间备份 |
| 每周 | weekly | 每周指定时间备份 |
| 每月 | monthly | 每月指定时间备份 |

**M7.4 备份执行时机**
1. **定时备份**: 按照配置的频率自动执行
2. **启动备份**: 程序启动时检查并执行
3. **关闭备份**: 程序关闭时检查并执行
4. **手动备份**: 用户手动触发备份

#### 2.3.2 配置界面

- 状态概览面板

- 高级设置面板

---

### 2.4 备份管理

#### 2.4.1 备份文件
- 格式: ZIP压缩包
- 命名: `backup_YYYYMMDD_HHMMSS.zip`
- 包含: 数据库文件 + 元数据

#### 2.4.2 备份元数据
```json
{
  "backup_type": "自动/手动",
  "backup_time": "2026-06-22_14:30:00",
  "app_version": "1.2.1",
  "task_count": 100,
  "contact_count": 50
}
```

#### 2.4.3 备份存储
- 默认位置: `backups/` 目录
- 支持自定义备份路径
- 自动创建目录（如果不存在）

---

### 2.5 旧备份清理

#### 2.5.1 清理策略
- 按保留数量清理
- 按配置保留最近N个备份
- 清理时保留手动备份

#### 2.5.2 清理时机
- 每次备份完成后自动清理
- 仅清理自动备份
- 手动备份不受影响

### 2.7 技术实现

#### 2.7.1 模块结构
```
src/
└── core/
    └── auto_backup_service.py     # 自动备份服务
└── ui/
    └── widgets/
        └── auto_backup_config_widget.py  # 配置界面
```

#### 2.7.2 核心类

**BackupFrequency (枚举)**
```python
class BackupFrequency(Enum):
    EVERY_HOUR = "hourly"        # 每小时
    EVERY_4_HOURS = "4hours"     # 每4小时
    EVERY_8_HOURS = "8hours"     # 每8小时
    DAILY = "daily"              # 每天
    WEEKLY = "weekly"            # 每周
    MONTHLY = "monthly"          # 每月
```

**AutoBackupConfig**
```python
class AutoBackupConfig:
    - DEFAULT_CONFIG: Dict           # 默认配置
    - db: DatabaseConnection         # 数据库连接
    - config: Dict                    # 当前配置
    - get(key, default) -> any        # 获取配置
    - set(key, value)                 # 设置配置
    - save() -> bool                  # 保存配置
    - frequency_seconds -> int         # 频率秒数
    - should_backup_now() -> bool      # 是否需要备份
    - get_next_backup_time() -> datetime  # 下次备份时间
```

**AutoBackupService**
```python
class AutoBackupService:
    - _instance: AutoBackupService    # 单例实例
    - config: AutoBackupConfig        # 备份配置
    - backup_service: BackupService   # 备份服务
    - _timer: threading.Timer         # 定时器
    - _running: bool                  # 运行状态
    - start()                         # 启动服务
    - stop()                          # 停止服务
    - trigger_backup(reason)          # 触发备份
    - get_status() -> Dict            # 获取状态
    - update_config(config)           # 更新配置
```
