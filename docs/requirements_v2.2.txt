## 1. 功能概述

### 1.1 功能名称
**自动备份功能** (M7.数据管理)

### 1.2 功能目标
为市场咨询任务跟踪工具提供自动化的数据备份能力，确保业务数据的安全性和可恢复性。

### 1.3 功能特性
- 定时自动备份
- 可配置的备份策略
- 旧备份自动清理
- 启动/关闭时自动备份
- 备份状态可视化

---

## 2. 功能需求

### 2.1 核心功能

#### 2.1.1 自动备份服务
- 单例模式运行的后台服务
- 支持多种备份频率
- 后台定时执行，不影响主程序运行
- 自动判断是否需要执行备份

#### 2.1.2 备份策略配置
| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| enabled | bool | false | 是否启用自动备份 |
| frequency | enum | daily | 备份频率 |
| backup_time | string | "02:00" | 每日备份时间 |
| keep_count | int | 7 | 保留备份数量 |
| backup_on_startup | bool | true | 启动时备份 |
| backup_on_shutdown | bool | true | 关闭时备份 |
| notify_on_backup | bool | true | 备份完成通知 |

#### 2.1.3 备份频率选项
| 频率 | 枚举值 | 说明 |
|------|--------|------|
| 每小时 | hourly | 每隔1小时备份一次 |
| 每4小时 | 4hours | 每隔4小时备份一次 |
| 每8小时 | 8hours | 每隔8小时备份一次 |
| 每天 | daily | 每天指定时间备份 |
| 每周 | weekly | 每周指定时间备份 |
| 每月 | monthly | 每月指定时间备份 |

#### 2.1.4 备份执行时机
1. **定时备份**: 按照配置的频率自动执行
2. **启动备份**: 程序启动时检查并执行
3. **关闭备份**: 程序关闭时检查并执行
4. **手动备份**: 用户手动触发备份

### 2.2 配置管理

#### 2.2.1 配置持久化
- 配置存储在数据库表中
- 支持运行时修改配置
- 配置变更自动生效

#### 2.2.2 配置界面
提供图形化配置界面，包括：
- 状态概览面板
- 基本设置面板
- 备份策略面板
- 高级设置面板

### 2.3 备份管理

#### 2.3.1 备份文件
- 格式: ZIP压缩包
- 命名: `backup_YYYYMMDD_HHMMSS.zip`
- 包含: 数据库文件 + 元数据

#### 2.3.2 备份元数据
```json
{
  "backup_type": "自动/手动",
  "backup_time": "2026-06-22_14:30:00",
  "app_version": "1.2.1",
  "task_count": 100,
  "contact_count": 50
}
```

#### 2.3.3 旧备份清理
- 自动删除超出保留数量的旧备份
- 优先删除最早的备份
- 同步删除数据库记录

## 4. 技术实现

### 4.1 模块结构

```
src/
├── core/
│   └── auto_backup_service.py    # 自动备份服务
├── ui/
│   └── widgets/
│       └── auto_backup_config_widget.py  # 配置界面
└── main.py                      # 程序入口
```

### 4.2 核心类

#### AutoBackupService
```python
class AutoBackupService:
    def start()                    # 启动自动备份
    def stop()                     # 停止自动备份
    def trigger_backup(reason)     # 手动触发备份
    def get_status()               # 获取状态
    def update_config(config)      # 更新配置
```

#### AutoBackupConfig
```python
class AutoBackupConfig:
    def get(key, default)          # 获取配置
    def set(key, value)            # 设置配置
    def save()                     # 保存配置
    def should_backup_now()        # 检查是否需要备份
    def get_next_backup_time()     # 获取下次备份时间
```

### 4.3 数据库表

```sql
CREATE TABLE auto_backup_config (
    id INTEGER PRIMARY KEY CHECK (id = 1),
    config_data TEXT NOT NULL,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
```
