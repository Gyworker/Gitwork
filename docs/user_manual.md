## 产品介绍

### 概述

市场咨询任务跟踪工具是一款面向市场咨询团队的任务管理应用，帮助团队高效管理咨询任务、智能推荐解决方案、跟踪任务进度。

### 主要功能

- **任务管理**：创建、编辑、删除、跟踪咨询任务
- **智能推荐**：基于关键字智能推荐历史解决方案
- **通讯录管理**：管理客户和团队联系人（含OCR扫描）
- **数据导入导出**：支持Excel/CSV/MSG邮件格式
- **OCR识别**：从名片/文档图片识别联系人信息
- **统计分析**：多维度统计任务分布情况
- **提醒通知**：任务到期提醒

### 系统要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10/11, macOS 10.14+ |
| 内存 | 4GB RAM以上 |
| 硬盘 | 500MB可用空间 |
| Python | 3.10+ |
| OCR引擎 | Tesseract 5.0+（可选） |

## 安装指南

### 方法一：从源码安装

```bash
# 1. 克隆仓库
git clone https://github.com/Gyworker/Gitwork.git
cd Gitwork

# 2. 创建虚拟环境
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

# 3. 安装依赖
pip install -r requirements.txt

# 4. 运行应用
python src/main.py
```

### 方法二：OCR功能安装（可选）

如需使用OCR名片识别功能，需要额外安装：

1. 下载Tesseract OCR引擎：https://github.com/UB-Mannheim/tesseract/wiki
2. 安装时选择中文语言包（chi_sim+eng）
3. 确保tesseract可执行文件在系统PATH中

## 快速入门

### 1. 创建第一个任务

1. 点击主界面 **"新建任务"** 按钮
2. 填写任务信息：
   - **标题**：任务名称（必填）
   - **描述**：任务详细描述
   - **重要性**：高/中/低
   - **责任人**：选择或输入责任人
   - **截止日期**：设置任务截止时间
3. 点击 **"保存"** 按钮

### 2. 跟踪任务进度

1. 在任务列表中选择要跟踪的任务
2. 点击 **"编辑"** 更新任务状态
3. 状态可选：待处理 → 进行中 → 已完成

### 3. 使用智能推荐

1. 创建或编辑任务时
2. 在推荐区域输入关键字
3. 系统自动显示相关历史任务

1. 打开 **"通讯录"** 页面
2. 点击 **"添加"** 按钮
3. 选择 **"📝 手动输入"** 模式

   - 姓名（必填）
   - 工号
   - 部门
   - 职位
   - 电话
   - 邮箱

详见下一章节：[OCR扫描功能](#ocr扫描功能)

### 数据导入导出

#### 导出任务

1. 在任务列表页面
2. 点击 **"导出"** 按钮
3. 选择导出格式（Excel/CSV）
4. 选择导出范围（全部/筛选结果）
5. 选择保存路径
6. 点击 **"导出"**

#### 导入任务

1. 点击 **"导入"** 按钮
2. 选择导入文件
3. 预览导入数据
4. 确认导入

支持从Outlook MSG文件导入邮件内容：

1. 点击 **"导入"** → **"MSG邮件"**
2. 选择MSG文件
3. 系统自动解析邮件内容
4. 预览并确认导入

统计页面提供以下分析：

- **任务状态分布**：饼图展示各状态任务数量
- **重要性分布**：柱状图展示重要性分布

## OCR扫描功能

### 功能介绍

OCR扫描功能可以从名片、宣传册、会议资料等图片中自动识别联系人信息，支持提取：姓名、电话、邮箱、部门、职位等字段。

### 支持的图片格式

- PNG
- JPG / JPEG
- BMP
- GIF
- TIFF
- WebP

### 操作步骤

#### 1. 打开OCR扫描

1. 打开 **"通讯录"** 页面
2. 点击 **"添加"** 按钮
3. 选择 **"📷 OCR扫描"** 模式

#### 2. 选择图片

1. 点击 **"浏览..."** 按钮
2. 在文件选择对话框中，选择要识别的图片文件
3. 图片路径会显示在输入框中

#### 3. 开始识别

1. 点击 **"🔍 开始OCR识别"** 按钮
2. 系统开始处理图片
3. 识别完成后，下方会显示识别结果预览

#### 4. 查看结果

识别结果包含以下内容：

| 项目 | 说明 |
|------|------|
| 识别状态 | 成功/失败 |
| 识别耗时 | 处理时间（毫秒） |
| 提取信息 | 姓名、电话、邮箱、部门、职位 |
| 识别置信度 | 信息可信度百分比 |
| 原始文本 | OCR识别的原始文字 |

#### 5. 应用结果

1. 确认识别结果无误后
2. 点击 **"✓ 应用识别结果"** 按钮
3. 系统自动将识别结果填充到表单
4. 如有误差，可手动修改
5. 点击 **"保存"** 完成添加

### 示例

#### 名片识别示例

**原始名片图片**

```
┌─────────────────────────────┐
│                             │
│   张三                       │
│   产品经理                   │
│                             │
│   ☎ 13800138000             │
│   ✉ zhangsan@example.com    │
│                             │
│   XX科技有限公司             │
│   市场部                     │
│                             │
└─────────────────────────────┘
```

**识别结果**

| 字段 | 识别结果 | 置信度 |
|------|----------|--------|
| 姓名 | 张三 | 95% |
| 电话 | 13800138000 | 98% |
| 邮箱 | zhangsan@example.com | 96% |
| 公司 | XX科技有限公司 | 85% |
| 部门 | 市场部 | 90% |
| 职位 | 产品经理 | 88% |

### 注意事项

1. **图片质量**：请使用清晰的名片或文档图片，避免模糊、过暗或过曝
2. **语言支持**：支持中文和英文识别，建议安装chi_sim+eng语言包
3. **信息核实**：OCR识别结果仅供参考，请务必核对后使用
4. **复杂版面**：对于复杂排版的名片，建议手动输入

### 常见问题

**Q: OCR识别失败怎么办？**

A: 请检查以下几点：
- 图片是否清晰可读
- Tesseract OCR引擎是否正确安装
- 图片格式是否支持（PNG/JPG/BMP/GIF/TIFF/WebP）

**Q: 识别结果不准确怎么办？**

A: OCR识别结果仅供参考，请手动核对并修正后再保存。

**Q: 如何安装Tesseract OCR？**

A:
1. 访问 https://github.com/UB-Mannheim/tesseract/wiki
2. 下载Windows安装包
3. 运行安装程序
4. 选择安装中文语言包（chi_sim+eng）
5. 确保tesseract可执行文件在系统PATH中

## 常见问题

### Q1: 如何备份数据？

数据存储在SQLite数据库文件中，位于应用数据目录。定期备份该文件即可。

### Q2: 忘记保存密码了怎么办？

本应用不保存用户密码，无需登录。

### Q3: 导入数据格式有误怎么办？

系统会在导入时进行格式校验，错误的行会被标记。修正后重新导入即可。

### Q4: 如何升级到新版本？

1. 备份现有数据
2. 下载新版本
3. 替换程序文件
4. 启动新版本

### Q5: 统计数据不准确怎么办？

检查任务数据是否完整，删除错误数据后重新统计。

### Q6: OCR功能无法使用？

请确认：
1. Tesseract OCR引擎已安装
2. 系统PATH中包含tesseract路径
3. 已安装pytesseract和Pillow库
