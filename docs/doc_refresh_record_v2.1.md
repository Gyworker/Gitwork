## 新增功能说明

### 1. 工号字段支持

**解决的问题**：通讯录中存在姓名相同的联系人，需要工号区分

**实现内容**：
- tasks表新增 `inquirer_employee_id`, `respondent_employee_id` 字段
- learned_contacts表新增 `employee_id` 字段
- recommendation_learning表新增 `employee_id` 字段
- 唯一约束：`姓名 + 工号` 组合唯一

### 2. 数据刷新逻辑

**解决的问题**：新数据应该覆盖旧数据，而非保留旧数据

**实现内容**：
- 新数据有值 → 使用新数据
- 新数据为空 → 保留旧数据
- 各字段独立刷新

### 3. 推荐库多模块汇总

**解决的问题**：同一人擅长多个模块，需要正确汇总和匹配

**实现内容**：
- 新增 `all_modules` 字段汇总同一人所有关键模块
- 新增 `get_aggregated_recommendations()` 汇总视图API
- 三层匹配：key_module → related_modules → all_modules
