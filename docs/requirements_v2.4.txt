## 2. M8.系统功能 - 主题切换功能

### 2.1 功能概述

#### 2.1.1 功能名称
**主题切换功能** (M8.系统功能)

#### 2.1.2 功能目标
为市场咨询任务跟踪工具提供主题切换功能，允许用户根据个人喜好选择浅色或深色主题，提升用户体验和可访问性。

#### 2.1.3 功能特性
- 浅色主题和深色主题切换
- 主题配置持久化保存
- 实时预览主题效果
- 快捷键快速切换
- 完整的样式覆盖

---

### 2.2 功能需求

#### 2.2.1 核心功能

**M8.1 主题管理服务**
- 单例模式运行的后台服务
- 支持浅色和深色主题
- 主题配置持久化存储
- 完整的QSS样式表生成
- 运行时主题切换

**M8.2 主题类型**
| 主题 | 枚举值 | 说明 |
|------|--------|------|
| 浅色主题 | light | 白色背景、蓝色主色调，适合白天使用 |
| 深色主题 | dark | 深色背景、浅色文字，适合夜间使用 |

**M8.3 主题配置项**
| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| theme_type | string | "light" | 主题类型 |
| font_size | int | 10 | 字体大小 |
| font_family | string | "Microsoft YaHei" | 字体 |
| window_opacity | float | 1.0 | 窗口透明度 |
| animations_enabled | bool | true | 动画开关 |

**M8.4 主题颜色配置**
| 颜色类别 | 浅色主题 | 深色主题 |
|----------|----------|----------|
| 主色调 | #1976D2 | #90CAF9 |
| 背景色 | #FFFFFF | #121212 |
| 文字色 | #212121 | #FFFFFF |
| 边框色 | #E0E0E0 | #424242 |
| 成功色 | #4CAF50 | #81C784 |
| 警告色 | #FF9800 | #FFB74D |
| 错误色 | #F44336 | #E57373 |

### 2.5 技术实现

#### 2.5.1 模块结构
```
src/
└── core/
    └── theme_manager.py           # 主题管理器
└── ui/
    └── widgets/
        └── theme_config_widget.py  # 主题配置界面
```

#### 2.5.2 核心类

**ThemeManager**
```python
class ThemeManager:
    - 单例模式
    - apply_theme(theme_type): 应用主题
    - toggle_theme(): 切换主题
    - generate_stylesheet(): 生成QSS样式表
    - get_current_theme(): 获取当前主题
    - get_colors(): 获取主题颜色
    - set_font_size(size): 设置字体大小
```

**ThemeConfig**
```python
class ThemeConfig:
    - 配置管理
    - 保存到JSON文件
    - get(key, default): 获取配置
    - set(key, value): 设置配置
    - save(): 保存配置
```

**ThemeColors**
```python
class ThemeColors:
    - LIGHT: 浅色主题颜色配置
    - DARK: 深色主题颜色配置
```
