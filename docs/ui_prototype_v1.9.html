👥 通讯录数据结构说明

📊 数据库存储信息

存储位置: ./data/contacts.db 表名: contacts CREATE TABLE contacts ( id INTEGER PRIMARY KEY AUTOINCREMENT, name VARCHAR(50) NOT NULL, -- 姓名（必填） employee_id VARCHAR(20), -- 工号 phone VARCHAR(20), -- 手机号 email VARCHAR(100), -- 邮箱地址 department VARCHAR(50), -- 所属部门 position VARCHAR(50), -- 职位 created_at DATETIME, -- 创建时间 updated_at DATETIME -- 更新时间 );

📥 导入格式说明

格式1：Excel表格
列名需包含"姓名"，其他列自动识别
格式2：TXT文本
支持逗号、空格、Tab分隔
格式3：张三/02733（姓名/工号）
识别结果：姓名=张三，工号=02733
