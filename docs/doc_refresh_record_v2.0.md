#### 2.2.2 核心设计点

**整体架构**：
```
UI层：LearningWidget（学习积累界面）
    │
业务逻辑层：LearningService（综合学习服务）
    ├── ContactLearningService（通讯录学习服务）
    └── RecommendationLearningService（推荐库学习服务）
    │
数据访问层：数据库表（5个学习相关表）
```

**服务类设计**：

1. **ContactLearningService**（~300行）
   - learn_from_task()：从任务学习
   - _learn_consultant()：学习咨询者
   - _learn_respondent()：学习答复人
   - search_contacts()：搜索联系人
   - export_to_contacts()：导出到通讯录
   - get_statistics()：获取统计

2. **RecommendationLearningService**（~300行）
   - learn_from_task()：从任务学习
   - extract_modules()：提取关键模块
   - _learn_module_respondent()：学习模块-答复人关联
   - _learn_module_correlation()：学习模块关联
   - _learn_industry_module()：学习行业-模块映射
   - get_related_modules()：获取相关模块
   - export_to_recommendations()：导出到推荐库

3. **LearningService**（~100行）
   - learn_from_task()：统一学习入口
   - batch_learn_from_tasks()：批量学习
   - get_all_statistics()：获取所有统计
   - get_learning_service()：单例获取

**模块提取算法**：
- 支持分隔符：、`/`,`、`，`空格、`\n`、`\t`、`;`
- 预定义关键词库：路由协议、安全策略、网络设备等
- 关键词优先级：OSPF(10)、BGP(10)、防火墙(9)等

**置信度计算**：
- 1次：0.5
- 2-3次：0.6
- 4-5次：0.7
- 6-10次：0.8
- 10次以上：0.9
