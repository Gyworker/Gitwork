#### 部署方式

1. **源码部署**
   ```bash
   git clone https://github.com/Gyworker/Gitwork.git
   cd Gitwork
   pip install -r requirements.txt
   python src/main.py
   ```

2. **虚拟环境部署**（推荐）
   ```bash
   python -m venv venv
   source venv/bin/activate  # Windows: venv\Scripts\activate
   pip install -r requirements.txt
   python src/main.py
   ```
