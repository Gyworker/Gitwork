## 一、项目概述

### 1.1 项目名称
市场咨询任务跟踪工具

### 1.2 项目目标
开发一款面向市场咨询场景的任务跟踪管理工具，实现任务信息解析、记录、查询、跟踪、提醒的一体化管理，提升市场咨询工作的效率和质量。

### 1.3 运行环境
- **操作系统**：Windows
- **技术栈**：Python + PyQt5
- **数据库**：SQLite（本地存储）
- **打包形式**：可执行程序（.exe）

### 1.4 技术集成
| 集成项 | 实现方式 | 说明 |
|--------|----------|------|
| OCR识别 | 本地Tesseract OCR（优先）+ 预留云端API接口 | V1.6新增 |
| Outlook邮件 | 直接连接Outlook读取 + .msg文件导入 | V1.7新增 |
| MSG邮件文件 | extract-msg库解析 + 无第三方库备用方案 | V1.8新增 |
| 企业微信 | 自建应用发消息 + webhook群机器人 | V1.7新增 |
| 数据导出 | Excel表格 | V1.5新增 |
| 自动备份 | 定时自动备份 + 启动/关闭备份 | V2.1新增 |
| 主题切换 | 浅色/深色主题 + 字体大小可调 | V2.1新增 |

## 二、功能需求

### 2.1 内容导入功能

#### 2.1.1 功能说明
支持多种渠道的内容导入，自动解析为任务信息。

#### 2.1.2 导入方式
| 方式 | 说明 | 优先级 | 状态 |
|------|------|--------|------|
| 文本粘贴 | 直接粘贴文字内容 | P0 | ✅已实现 |
| 图片导入 | OCR识别（本地Tesseract优先，支持可视化切换按钮切换至云端API） | P0 | ✅已实现 |
| MSG邮件文件 | 读取导出的.msg文件，解析邮件内容 | P0 | ✅已实现 |
| 企业微信 | 聊天记录文件导入 + 截图OCR识别 | P1 | ✅已实现 |
| Outlook邮件 | 直接连接Outlook指定邮件读取 | P0 | ⚠️暂缓 |

#### 2.1.3 MSG邮件文件导入功能

##### 2.1.3.1 功能概述
支持导入Outlook导出的MSG格式邮件文件，自动解析邮件内容并提取任务相关信息。

##### 2.1.3.2 技术实现
| 组件 | 实现方式 | 说明 |
|------|----------|------|
| 首选方案 | extract-msg库 | 功能完整，解析准确 |
| 备用方案 | 自定义解析器 | 无第三方库时使用 |
| 文件大小限制 | 100MB | 防止大文件导致内存溢出 |

##### 2.1.3.3 MSG邮件解析字段
| 字段 | 说明 | 来源 |
|------|------|------|
| 任务名称 | 邮件主题或前50字符 | subject |
| 任务具体内容 | 邮件正文 | body |
| 咨询者姓名 | 发件人姓名 | sender_name |
| 咨询者邮箱 | 发件人邮箱地址 | sender_email |
| 咨询时间 | 邮件发送时间 | date |
| 附件信息 | 附件列表及文件名 | attachments |

##### 2.1.3.4 流式解析
- 支持大文件流式读取
- 批量导入时支持进度回调
- 支持取消导入操作

#### 2.1.4 解析结果编辑
- 导入后自动显示解析结果
- 支持手动修正解析内容
- 一键创建任务或保存草稿

### 2.11 自动备份功能（V2.1新增）

#### 2.11.1 功能概述
提供定时自动备份功能，支持多种备份频率和策略，确保数据安全。

#### 2.11.2 备份频率
| 频率 | 说明 | 适用场景 |
|------|------|----------|
| 每小时 | 每隔1小时备份一次 | 高数据安全性需求 |
| 每4小时 | 每隔4小时备份一次 | 中等数据安全性需求 |
| 每8小时 | 每隔8小时备份一次 | 标准数据安全性需求 |
| 每天 | 每天指定时间备份 | 默认选项 |
| 每周 | 每周指定日期备份 | 低数据变更频率 |
| 每月 | 每月指定日期备份 | 归档备份 |

#### 2.11.3 生命周期备份
| 触发条件 | 说明 | 默认值 |
|----------|------|--------|
| 启动时备份 | 程序启动时自动执行备份 | 启用 |
| 关闭时备份 | 程序关闭时自动执行备份 | 启用 |

#### 2.11.4 备份策略
| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 保留数量 | 自动清理超过保留数量的旧备份 | 7个 |
| 最大备份大小 | 单个备份文件的最大大小 | 100MB |
| 压缩级别 | ZIP压缩级别（0-9） | 6 |
| 备份通知 | 备份完成时发送系统通知 | 启用 |
