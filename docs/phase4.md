## 使用方法

### 运行所有测试
```bash
cd D:\GITwork
python -m pytest src/tests/ -v
```

### 运行快速测试
```bash
python -m pytest src/tests/ -v -m "not slow"
```

### 运行性能测试
```bash
python -m pytest src/tests/test_performance.py -v -s
```

### 运行特定测试
```bash
# 只测试数据库
python -m pytest src/tests/test_database.py -v

# 只测试核心模块
python -m pytest src/tests/test_core.py -v
```
