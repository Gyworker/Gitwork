## 2. 项目概述

### 2.1 项目背景

市场咨询任务跟踪工具是一款基于PyQt5开发的Windows桌面应用程序，用于帮助企业市场部门高效管理和跟踪客户咨询任务，提升客户服务质量和响应效率。

### 2.2 项目目标

- 提供直观易用的任务管理界面
- 支持多种数据导入方式（文本、图片、MSG、Outlook、企业微信）
- 实现智能任务推荐和应答辅助
- 支持数据自动备份和统计报表

## 3. 功能需求

### 3.1 核心功能模块

| 模块 | 功能 | 优先级 | 状态 |
|------|------|--------|------|
| 任务信息管理 | 创建、编辑、删除、查询任务 | P0 | ✅ 已完成 |
| 任务跟踪 | 跟踪任务状态，记录应答进度 | P0 | ✅ 已完成 |
| 智能推荐 | 基于历史数据推荐相似任务 | P1 | ✅ 已完成 |
| 提醒管理 | 设置任务提醒，逾期预警 | P1 | ✅ 已完成 |
| 数据导入 | 多种方式导入任务数据 | P0 | ✅ 已完成 |
| 统计分析 | 统计任务数据，生成报表 | P1 | ✅ 已完成 |
| 推荐库 | 管理标准回复和知识库 | P1 | ✅ 已完成 |
| 自动备份 | 自动备份数据库 | P2 | ✅ 已完成 |
| 主题切换 | 支持浅色/深色主题 | P2 | ✅ 已完成 |
| **智能学习&应答** | **基于关键模块匹配相似任务，提供答复参考** | **P1** | **✅ 已完成** |

### 3.2 数据导入功能（V2.5更新）

#### 3.2.1 导入方式

| 导入方式 | 说明 | 状态 |
|----------|------|------|
| 文本粘贴 | 直接粘贴或输入文本内容 | ✅ 已完成 |
| 图片导入 | 导入图片并使用OCR识别文字 | ✅ 已完成 |
| MSG邮件文件 | 导入Outlook MSG格式邮件 | ✅ 已完成（融合到内容导入） |
| Outlook邮件 | 导入Outlook邮件内容 | ✅ 已完成 |
| 企业微信 | 导入企业微信消息 | ✅ 已完成 |

#### 3.2.2 界面设计要求

- **主界面**：去除独立的"MSG导入"按钮
- **内容导入区域**：MSG邮件导入作为"导入方式"的一种
- **导入方式排序**：文本粘贴 → 图片导入 → MSG邮件文件 → 企业微信 → Outlook邮件

### 3.3 智能学习&应答功能（V2.5更新）

#### 3.3.1 功能描述

| 功能项 | 说明 |
|--------|------|
| 添加记录 | 支持txt文本、excel表格、图片(OCR识别)三种文档输入方式 |
| 任务关联 | 与任务信息页面联动，选中单个任务时可用，多选时报错 |
| 智能匹配 | 根据关键模块提供类似的多个任务信息（带匹配度显示） |
| 学习统计 | 记录总数、本月新增、关联任务数、待审核统计 |
| 记录管理 | 学习记录列表、导出、清空功能 |

#### 3.3.2 界面设计要求

- **导航标签**：🧠 智能学习&应答
- **工具栏按钮**：🧠 智能学习&应答（蓝色主题）
- **按钮颜色**：#1565C0（蓝色）
- **描述文本**："基于关键模块智能匹配相似任务，提供**答复参考**"

#### 3.3.3 多选任务处理

| 场景 | 处理方式 |
|------|----------|
| 选中0个任务 | 显示空状态，提示选择任务 |
| 选中1个任务 | 正常进入智能学习页面，关联任务 |
| 选中2个及以上任务 | 点击导航时报错："智能学习&应答功能仅支持关联单个任务" |
