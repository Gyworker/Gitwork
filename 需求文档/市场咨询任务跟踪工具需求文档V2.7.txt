## 2. 项目概述

### 2.1 项目背景

市场咨询任务跟踪工具是一款基于PyQt5开发的Windows桌面应用程序，用于帮助企业市场部门高效管理和跟踪客户咨询任务，提升客户服务质量和响应效率。

### 2.2 项目目标

- 提供直观易用的任务管理界面
- 支持多种数据导入方式（文本、图片、MSG、Outlook、企业微信）
- 实现智能任务推荐和应答辅助
- 支持数据自动备份和统计报表

## 3. 功能需求

### 3.1 核心功能模块

| 模块 | 功能 | 优先级 | 状态 |
|------|------|--------|------|
| 任务信息管理 | 创建、编辑、删除、查询任务 | P0 | ✅ 已完成 |
| 任务跟踪 | 跟踪任务状态，记录应答进度 | P0 | ✅ 已完成 |
| 智能推荐 | 基于历史数据推荐相似任务 | P1 | ✅ 已完成 |
| 提醒管理 | 设置任务提醒，逾期预警 | P1 | ✅ 已完成 |
| 数据导入 | 多种方式导入任务数据 | P0 | ✅ 已完成 |
| 统计分析 | 统计任务数据，生成报表 | P1 | ✅ 已完成 |
| 推荐库 | 管理标准回复和知识库 | P1 | ✅ 已完成 |
| 自动备份 | 自动备份数据库 | P2 | ✅ 已完成 |
| 主题切换 | 支持浅色/深色主题 | P2 | ✅ 已完成 |
| **智能学习&应答** | **基于关键模块匹配相似任务，提供答复参考** | **P1** | **✅ 已完成** |

### 3.2 数据导入功能

#### 3.2.1 导入方式

| 导入方式 | 说明 | 状态 |
|----------|------|------|
| 文本粘贴 | 直接粘贴或输入文本内容 | ✅ 已完成 |
| 图片导入 | 导入图片并使用OCR识别文字 | ✅ 已完成 |
| MSG邮件文件 | 导入Outlook MSG格式邮件 | ✅ 已完成（融合到内容导入） |
| Outlook邮件 | 导入Outlook邮件内容 | ✅ 已完成 |
| 企业微信 | 导入企业微信消息 | ✅ 已完成 |

#### 3.2.2 界面设计要求（V2.12优化）

**导入方式区域设计**：

| 导入方式 | 输入区域 | 说明 |
|----------|----------|------|
| 文本粘贴 | 文本输入框 | 直接输入或粘贴文本内容 |
| 图片导入 | 文件上传区 | 上传图片进行OCR识别 |
| MSG邮件 | 文件上传区 | 上传MSG邮件文件解析 |
| 企业微信 | 输入区 | Webhook URL + 消息内容输入 |
| Outlook | 输入区 | 账户选择 + 邮件筛选输入 |

**企业微信输入区要求**：
- Webhook URL输入框：企业微信机器人Webhook地址
- 消息内容输入框：要发送的消息内容文本框
- 布局：上下排列，标签+输入框

**Outlook输入区要求**：
- 账户选择下拉框：选择要读取的邮箱账户
- 邮件筛选输入框：输入发件人或主题关键词
- 布局：上下排列，标签+输入框

**统一内容预览区要求**：
- 所有导入模式统一使用预览区显示解析结果
- 点击"解析"按钮后，解析结果自动显示在预览区
- 预览区样式：绿色边框表示有内容

**导入方式排序**：文本粘贴 → 图片导入 → MSG邮件文件 → 企业微信 → Outlook邮件

### 3.3 智能学习&应答功能

#### 3.3.1 功能描述

| 功能项 | 说明 |
|--------|------|
| 添加记录 | 支持txt文本、excel表格、图片(OCR识别)三种文档输入方式 |
| 任务关联 | 与任务信息页面联动，选中单个任务时可用，多选时报错 |
| 智能匹配 | 根据关键模块提供类似的多个任务信息（带匹配度显示） |
| 学习统计 | 记录总数、本月新增、关联任务数、待审核统计 |
| 记录管理 | 学习记录列表、导出、清空功能 |

#### 3.3.2 界面设计要求

- **导航标签**：🧠 智能学习&应答
- **工具栏按钮**：🧠 智能学习&应答（蓝色主题）
- **按钮颜色**：#1565C0（蓝色）
- **描述文本**："基于关键模块智能匹配相似任务，提供**答复参考**"

#### 3.3.3 多选任务处理

| 场景 | 处理方式 |
|------|----------|
| 选中0个任务 | 显示空状态，提示选择任务 |
| 选中1个任务 | 正常进入智能学习页面，关联任务 |
| 选中2个及以上任务 | 点击导航时报错："智能学习&应答功能仅支持关联单个任务" |

## 4. 界面优化需求

### 4.1 任务信息字段排序（V2.8优化）

**变更内容**：关键模块字段调整到答复人姓名之前

**调整后字段顺序**：
1. 任务名称
2. 咨询人信息（咨询人姓名、咨询人部门、咨询人公司）
3. **关键模块** ← 调整到这里
4. 任务具体内容
5. 紧急程度
6. **答复人信息**（答复人姓名、答复人部门）← 从这里移走
7. 行业
8. 状态
9. 预期答复时间
10. 备注

### 4.2 任务导出优化（V2.8优化）

**新增功能**：
- 支持自定义导出字段
- 包含处理时间计算（处理时间 = 答复时间 - 创建时间）
- 支持按状态导出（导出应答中、导出已答复）

### 4.3 通讯录导入增强（V2.8优化）

**新增功能**：
- 支持图片识别(OCR)导入方式
- 支持Excel表格导入
- 支持txt文本导入

### 4.4 通讯录重名处理（V2.8优化）

**处理规则**：
- 姓名相同时按工号区分
- 信息不一致时自动刷新最新信息

### 4.5 推荐库多映射（V2.8优化）

**功能说明**：
- 支持同一姓名对应多个关键模块
- 搜索推荐人时支持多关键模块匹配

### 4.6 表格列宽可调整（V2.8新增）

**功能说明**：
- 任务跟踪表格支持拖动列头分割线调整列宽
- 拖动分割线时显示宽度指示
- 调整后的列宽自动保存

### 4.7 智能学习翻页功能（V2.8新增）

**功能说明**：
- 选中任务后支持翻页查看更多相似任务
- 显示当前页码和总页数
- 支持跳转到指定页面

### 4.8 日志自动保存（V2.8新增）

**功能说明**：
- 日志内容超过10KB时自动保存为文件
- 保存路径：logs目录下
- 保留最新日志在界面显示

### 4.9 内容导入增强（V2.12新增）

**企业微信输入增强**：

| 字段 | 类型 | 说明 |
|------|------|------|
| Webhook URL | 文本输入 | 企业微信机器人Webhook地址 |
| 消息内容 | 多行文本 | 要发送的消息内容 |

**Outlook输入增强**：

| 字段 | 类型 | 说明 |
|------|------|------|
| Outlook账户 | 下拉选择 | 选择要读取的邮箱账户 |
| 邮件筛选 | 文本输入 | 发件人或主题关键词 |

**统一内容预览区**：

| 功能 | 说明 |
|------|------|
| 预览区域 | 所有导入模式的解析结果显示区 |
| 显示条件 | 点击"解析"按钮后显示解析结果 |
| 样式标识 | 绿色边框表示有内容 |
